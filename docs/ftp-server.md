# ftp-server
Simple FTP Server

## Usage
    ftp-server  [--host <hostname>]
                [--port <port>]
                [--user <username>]
                [--password <password>]
                [--dir <dirpath>]

    ftp-server
    ftp-server --port 50021

## Default Settings
- IP: 0.0.0.0
- port: 60021
- user: user
- password: password
- dir: $PWD
